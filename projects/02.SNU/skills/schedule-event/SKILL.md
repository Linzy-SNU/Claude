# Schedule Event (일정 등록)

서울대 학사일정에 맞춰 Google Calendar에 일정을 등록하는 스킬.

## 사용 시점

다음과 같은 요청에 자동으로 트리거:
- "일정 등록해줘"
- "캘린더에 추가해줘"
- "수업 일정 넣어줘"
- "시험 일정 등록"
- "모임 잡아줘"

## 워크플로

### 1단계: 어떤 일정인지 묻기

먼저 사용자에게 다음과 같이 물어본다:

> **어떤 일정을 등록할까요?**

사용자가 즉시 모든 정보를 주지 않으면, 다음 항목을 단계별로 수집:

| 항목 | 예시 | 필수 여부 |
|------|------|----------|
| 제목 | "영문학개론" | 필수 |
| 날짜/요일 | "매주 월요일" or "5월 15일" | 필수 |
| 시간 | "10:00~12:00" | 필수 |
| 장소 | "Building 26, Room 102" | 선택 |
| 반복 여부 | "이번 학기 매주" | 선택 |
| 메모/설명 | "교재: ..." | 선택 |

### 2단계: 자동 검증

수집한 정보로 다음을 자동 검증:

1. **학기 기간 체크**
   - 2026학년도 1학기: 2026-03-03(개강) ~ 2026-06-15(종강)
   - 반복 일정이면 종강일까지만 설정

2. **한국 공휴일 자동 제외**
   - 법정 공휴일 + 대체공휴일 + 임시공휴일
   - 2026년 학기 중 주요 휴일:
     - 5/5 어린이날 (화)
     - 5/24 부처님오신날 → 5/25(월) 대체공휴일
     - 6/3 지방선거 임시공휴일 (수)
     - 6/6 현충일 (토, 대체공휴일 없음)

3. **법정 공휴일 아닌 기념일은 제외하지 않음**
   - 스승의 날, 노동자의 날 등은 그대로 등록

### 3단계: 미리보기 + 확인

등록 전 사용자에게 보여주고 컨펌 받기:

```
다음 일정을 등록할까요?

📅 제목: [제목]
🕐 시간: [요일] [시간]
📍 위치: [장소]
🔁 반복: [반복 규칙]
🚫 제외: [공휴일 목록]

진행할까요?
```

**반드시 사용자 확인 후 등록한다.** 자동으로 등록하지 말 것.

### 4단계: Google Calendar 등록

`mcp__075e9375-...__create_event` 도구 사용:
- `calendarId`: primary (또는 사용자 지정)
- `timeZone`: Asia/Seoul
- 반복 일정은 RRULE 형식: `FREQ=WEEKLY;UNTIL=YYYYMMDDTHHMMSSZ;BYDAY=MO,TU,...`
- 위치 형식: `Seoul National University Building [X] | Room: [번호]` (강의실의 경우)

### 5단계: 결과 알리기

등록 후 결과 요약:
- 등록된 인스턴스 수
- 제외된 공휴일 (있다면)
- 캘린더 링크

## 후속 작업 옵션

등록 후 사용자에게 다음 옵션 제시:
- 일정 요약을 이메일로 보내기 (Gmail MCP 또는 Mac Mail 사용)
- 다른 일정도 추가하기

## 사용 도구

- `mcp__075e9375-...__create_event`: 일정 생성
- `mcp__075e9375-...__list_events`: 일정 조회
- `mcp__075e9375-...__delete_event`: 일정 삭제
- `mcp__075e9375-...__update_event`: 일정 수정
- `WebSearch`: 학사일정/공휴일 확인 시 사용
- `gmail-send` MCP: 일정 요약 이메일 발송

## 주의사항

- **확인 없이 자동 등록 금지** (사용자 컨펌 필수)
- **반복 일정은 학기 종강일까지만** (다음 학기는 수강신청 후 별도 등록)
- **공식 공휴일/대체공휴일/임시공휴일만 자동 제외** (스승의 날 등 비공식 기념일은 등록)
- **위치는 Google Maps와 호환되는 형식** 사용
